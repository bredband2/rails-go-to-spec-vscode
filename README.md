# Rails Go to Spec extension for VSCODE

Jump between code and spec in Rails projects.

To install search for

```
rails-go-to-spec-2
```

## Default keybinding:

- Ctrl + Shift + y
- Cmd + Shift + y (Mac)

## Redine shortcuts:

In keybindings.json

```
  ...
	{
		"key": "shift-cmd-y",
		"command": "rails-go-to-spec-2.railsGoToSpec",
		"when": "editorFocus"
	}
	...
```
